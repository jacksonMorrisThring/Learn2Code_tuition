Want to Learn2Code?

Learn2Code is a simple, regimented, 1 on 1 learning environment that focuses on school age kids, but can comfortablty teach university students/teachers as well.

Often times in big classes, students can easily be left behind by either the teachers inability, or the student struggeling to bring attention to their own needs among so many others. Learn2Code side steps this through its 1-on-1 style in the students own environment.

Why start so young?

The Human brain at younger ages has very high neural plasticity!

What this means is that children are experts at picking up new skills (like languages) much easier than an adult could. Computer coding is a language based skill, believe it or not. Building proficiency, or at least familiarity with computer code sets your child up for endless high paying career paths in anything from Software Development, to Science. Without firm coding skills, these paths become harder and harder to follow in a world being gripped by technology more and more everyday!

FAQs

How do you make computer coding accessible for me/my child?

We start at the very basics, using something called the Terminal to navigate and make a folder (just like using finder on mac, or windows document finder). Using this tool, and my guidence, we can compile and run code as well as make folders, navigate, and create new coding documents. We write the code in a 'text-editor' called sublime. Its easy to use, and nicely color coordinated to help reduce confusion. Using these tools, we can build some basic addition calculators. Slowely but surely, we build up complexity until we can use functions (tools to do our heavy lifting) and control statements.
